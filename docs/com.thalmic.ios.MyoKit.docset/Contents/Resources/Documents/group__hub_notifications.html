Detailed Description

These are events sent when the manager detects any change in a TLMMyo's state. These can be subscribed to through NSNotificationCenter. They are posted in the default center on the main dispatch queue.

Variable Documentation

NSString* const TLMHubDidAttachDeviceNotification
